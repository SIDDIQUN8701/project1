This is the left-arm of the website. The heart is flanked by two right and left columns. I was planning on using the columns as permanent objects on each page. They would be used to help the navigation across the site. For example, I could post links to the other pages of my website by using an list without the bullet points removed (list-style: none). Because they would be on every page, it would be easier to click on any one of the links and navigate directly to the page you want to visit. This column is made by using a flexbox and centering the content inside of it. It is given a width of 200 pixels and is not allowed to shrink by setting its "flex-shrink" to 0. Both columns and the heart are actually three different children in a flexbox set in the chester tag. The chester area is the middle area of the page that does not include the header or footer. It sets its children to align in a row and stretch to fit the page as well as fill the gaps inbetween each other.
